[Translate to English](https://github.com/sthefanyalaminos/dashboard_dados_com_python/blob/main/README_EN.md)

O Dashboard foi desenvolvido com o objetivo de realizar análise exploratória de dados de forma automatizada e visual, permitindo a interpretação rápida de informações estratégicas por meio de gráficos e filtros interativos.

## Objetivo: 
Aplicar os conceitos aprendidos durante as aulas, envolvendo: 
- Limpeza e preparação de dados;
- Aplicação de filtros simples;
- Extração de insights;
- Análise de estatísticas;
- Construção de visualizações interativas.


## Tecnologias utilizadas:
- Python;
- Pandas;
- Streamlit;
- Plotly.

## Metodologia de Análise
A análise dos dados foi conduzida em etapas estruturadas, para garantir organização, clareza e consistência nas informações apresentadas no dashboard.
1. Tratamento e Preparação dos Dados
- Limpeza de dados ausentes ou inconsistentes;
- Padronização de variáveis categóricas;
- Ajustes nos formatos de dados numéricos e textuais. 
2. Análise Exploratória
- Identificação de padrões nos salários;
- Análise da distribuição de cargos;
- Avaliação da relação entre modelo de trabalho e remuneração.
3. Visualização dos Dados
- Construção de gráficos interativos para facilitar a interpretação utilizando Plotly;
- Uso de filtros dinâmicos para comparação de cenários;
- Organização visual com foco em clareza e acessibilidade. 

## Melhorias Implementadas
Além da estrutura proposta durante a Imersão Dados com Python da Alura, foram realizadas melhorias com o objetivo de tornar o dashboard mais intuitivo, acessível e visualmente consistente.

### Padronização Visual
Foi aplicada uma identidade visual em tons de azul, buscando maior consistência estética e melhor experiência de leitura dos dados.
A padronização contribui para uma interface mais organizada, facilitando a interpretação das informações apresentadas.

###  Aprimoramento dos Filtros
A lógica de funcionamento dos filtros foi ajustada para tornar a navegação mais simples e intuitiva para o usuário:
- Quando não há nenhuma seleção, todos os dados são exibidos automaticamente.
- Ao selecionar um filtro específico, o dashboard apresenta apenas os dados daquele período / categoria.
- É possível selecionar múltiplos filtros, permitindo também comparações entre diferentes períodos.
- Ao remover todas as seleções, o dashboard retorna automaticamente à visualização completa dos dados.
Essa melhoria torna a exploração dos dados mais prática e flexível.

### Suporte a Múltiplos Idiomas
Foi implementada a opção de visualização do dashboard em português e inglês, permitindo que usuários de diferentes contextos possam explorar a aplicação. Essa funcionalidade amplia o alcance do projeto e melhora sua acessibilidade para um público internacional.
